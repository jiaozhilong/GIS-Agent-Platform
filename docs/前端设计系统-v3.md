# GIS-Agent 平台 — 前端设计系统 v3

> **依据**: UI 设计师此前交付的 `gis-login-v2.html`（龙哥确认方向）
> **状态**: 已确认基线，后续所有前端页面以此为规范

---

## 1. 配色

| 角色 | 色值 | 用途 |
|------|------|------|
| 底色 Base | `#060b14` | 页面最底层 |
| 面板 Panel | `rgba(10, 16, 30, 0.92)` | 侧栏/卡片/登录面板 |
| 主色 Primary | `#3b82f6`（蓝） | 主操作、激活态、品牌识别 |
| 辅色 Green | `#10b981`（绿） | 成功/运行中状态 |
| 辅色 Gold | `#f59e0b`（金） | 标注/城市光点 |
| 辅色 Cyan | `#06b6d4` | 数据流/次要强调 |
| 文字主 | `#e2e8f0` | 正文 |
| 文字次 | `#94a3b8` | 辅助说明 |
| 边框 | `rgba(59, 130, 246, 0.18)` | 卡片/输入框 |

## 2. 字体

`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`

| 层级 | 字号/行高 | 字重 | 用途 |
|------|----------|------|------|
| 品牌标题 | 28~40px / 1.25 | 700 | 品牌区大字 |
| 页面标题 | 26px | 700 | 面板标题 |
| 区块标题 | 15px | 600 | 卡片标题 |
| 正文 | 14px | 400 | 正文/表单 |
| 辅助 | 13px / 12px / 11px | 400 | 标签/说明/时间戳 |

## 3. 组件规范

- **按钮 Primary**：蓝色渐变 `#1d4ed8 → #3b82f6`，悬停 `#3b82f6 → #6366f1`，圆角 14px
- **按钮 Secondary**：透明底 + `rgba(59,130,246,0.18)` 边框，悬停亮蓝
- **卡片**：面板底色 `rgba(10,16,30,0.92)` + 边框 `rgba(59,130,246,0.18)`，圆角 14px
- **输入框**：暗底 `rgba(255,255,255,0.04)` + 边框，聚焦蓝光 `box-shadow: 0 0 0 3px rgba(59,130,246,0.25)`
- **状态标签**：rounded-full，状态色底 + 状态色文字
- **导航**：深色侧栏，激活项左侧蓝色边框

## 4. 登录页

- 左右分栏：左侧品牌区（3D 线框地球 + 品牌文字 + 底部数据指标），右侧登录表单面板
- 3D 球：Three.js，线框球体 + 大陆光点（绿）+ 城市标记（金）+ 数据弧线 + 轨道环 + 卫星粒子 + 大气辉光 + 星空
- 品牌标题渐变：`#60a5fa → #34d399 → #fbbf24`
- 底部数据指标：50+ 行业模板 / 12 可编排 Skill / 100% 本地运行
- 表单：用户名 + 密码 + 记住登录 + SSO 入口（企业微信/GitHub）
