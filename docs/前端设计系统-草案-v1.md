# GIS-Agent 平台 — 前端设计系统草案 v1

> **产出**: UI 设计师专家（PM 调度） | **状态**: 🔒 待龙哥确认 | **日期**: 2026-07-16
> **设计依据**: 平台特性 = 给 GIS 解决方案工程师用的 AI Agent 平台，基于 IMA 知识库 + 大模型，通过 9 工具流水线自动生成解决方案（需求→产品→案例→竞品→架构→大纲→质检→输出）。
> **设计原则**: 专业、智能、GIS 科技感。**不含预设"模型大爆炸"动效**——动效方向见第 5 节，由龙哥确认。

---

## 1. 设计调性

面向 GIS 解决方案工程师（技术型用户），平台核心是"知识库 + 大模型 → 自动生成方案"。视觉应传达：
- **专业可信**：信息密度高、结构清晰，像 GIS 指挥中心/控制台。
- **智能流动**：体现数据在 9 工具间流转的"流水线"感。
- **GIS 科技感**：暗色基底 + 空间数据可视化常用的高对比配色。

> 建议整体走**深色科技风**（具体见配色）。若龙哥更倾向浅色/其他方向，可在确认环节提出调整。

---

## 2. 配色规范

| 角色 | 色值 | 用途 |
|------|------|------|
| 底色 Base | `#0A1220` | 页面最底层（深蓝黑） |
| 表面 Surface | `#111C30` | 卡片/面板底色 |
| 抬升 Elevated | `#16243D` | 弹层/悬停态 |
| 主色 Primary | `#2DD4BF`（青/蓝绿） | 主操作、激活态、品牌识别——"智能" |
| 辅色 Secondary | `#38BDF8`（天蓝） | 数据流、链接、次要强调——"数据" |
| 成功 Success | `#34D399` | 流水线 SUCCESS |
| 警告 Warning | `#FBBF24` | PARTIAL / 需关注 |
| 危险 Danger | `#F87171` | FAILED |
| 文字主 | `#E6EDF6` | 正文 |
| 文字次 | `#94A3B8` | 辅助说明 |
| 边框 Border | `rgba(255,255,255,0.08)` | 卡片/分割线 |

**配色理由**：GIS 空间数据可视化（地图、遥感、控制台）惯用暗色底以突出数据层；青/蓝绿 + 天蓝 既科技又区分"智能主操作"与"数据流"，且对色弱友好。

## 3. 字体与字号

- **字体栈**：`"HarmonyOS Sans SC", "Inter", "PingFang SC", system-ui, sans-serif`（中文 HarmonyOS/苹方，西文 Inter）
- **等宽**（代码/JSON/工具输出）：`"JetBrains Mono", "Cascadia Code", monospace`

| 层级 | 字号/行高 | 字重 | 用途 |
|------|----------|------|------|
| H1 | 28 / 36 | 700 | 页面标题 |
| H2 | 22 / 30 | 600 | 区块标题 |
| H3 | 18 / 26 | 600 | 卡片标题 |
| Body | 14 / 22 | 400 | 正文 |
| Caption | 12 / 18 | 400 | 辅助/时间戳 |

## 4. 核心组件规范

- **按钮 Primary**：青色填充 `#2DD4BF`、深色文字、圆角 8、悬停提亮、点击微缩放。
- **卡片 Card**：Surface 底 `#111C30`、1px 半透明白边、圆角 12、轻微阴影；悬停抬升 + 边框转青。
- **导航 Sider**：左侧 220px 暗色栏，激活项青色左边框 + 底色微染青。
- **工具节点**（流程画布用）：圆角矩形，左侧图标 + 工具名 + 右侧状态点（青=运行 / 绿=成功 / 红=失败 / 灰=待执行）。
- **表格**：斑马纹用 Elevated 微差，表头底色 Surface，行悬停提亮。
- **标签 Tag**：rounded-full，状态色低饱和底 + 状态色文字。

## 5. 动效方向（三选一，由龙哥定；不再预设大爆炸）

动效从**平台特性**推导，非凭空设计：

| # | 概念 | 映射的系统特性 | 适用场景 |
|---|------|--------------|---------|
| **A** | **管线汇聚**：粒子从四周汇聚成一条流动的"工具链"光带，依次点亮 9 个节点 | 9 工具流水线生成方案的核心机制 | 登录页 / 首页 Hero |
| **B** | **知识图谱生长**：节点与连线逐步生长铺开，连线带数据流光 | IMA 知识库检索 + 大模型推理 | 登录页 / 加载态 |
| **C** | **地图网格点亮**：暗色地图网格按波纹依次点亮，叠加深蓝渐变 | GIS 空间底色 | 登录页背景 |

> 龙哥可三选一、组合，或否定全部另提方向。**确认前不动手实现动效。**
